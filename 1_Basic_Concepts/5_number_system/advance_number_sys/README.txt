#######Number datatypes in python########
Python support integers(int), floating(float) point numbers and complex numbers
Data type      Example
---------     ---------
int              5
float            5.9
complex          5+3j    5 is in real part and 3 in imaginary part

Floating point number is accurate only upto 15 decimal places(16 decimal place inaccurate)

Information about base system
Base             Explanation                            Prefix              Example            Conversion
----             -----------                            -------            ----------          ----------
base 10          The numbers we deal with every day                         4,5,7
base 2           Binary numbers                         '0b' or '0B'        print(0b1001)      bin(9)
base 8           Octal numbers                          '0o' or '0O'        print(0o1001)      oct(9)
base 16          Hexadecimal numbers                    '0x' or '0X'        print(0x1001)      hex(9)
